# 学生档案管理系统（试用版）

> 25 级汽修一班学生档案管理系统 · Tkinter 桌面应用  
> 适用于：Windows（推荐），也可在具备桌面环境的 Linux / macOS 上运行。

---

## 功能概览

- **Excel 导入/导出**：批量导入学生档案，支持选择列导出，可导出筛选结果或全部数据。  
- **列管理**：支持新增列、删除列、列分类（按大类筛选列、配置列归属）。  
- **查询筛选**：
  - 姓名模糊搜索。
  - 按列大类 → 列名 → 列值三级筛选。
  - 清空筛选后恢复全量数据。  
- **详情编辑**：双击学生记录打开详情，分标签页按列大类展示，可直接编辑并保存。  
- **数据库持久化**：数据存储在本地 SQLite（`student_database.db`），避免 Excel 导入导出造成的信息丢失。  
- **状态提示**：状态栏实时显示当前数据量、筛选状态、导入导出进度等。

## 快速开始

1. **克隆/下载代码**

   ```bash
   git clone <your-repo-url>
   cd <project-folder>


   常见操作指南
1. 导入学生 Excel
点击 导入Excel。
选择含 学号 列的 Excel 文件（支持 .xlsx/.xls）。
若列已存在会提示覆盖，新增列会提示确认。
完成后自动刷新界面。
2. 导出学生 Excel
点击 导出Excel。
若当前处于筛选状态，可选择导出“筛选结果”或“全部数据”。
勾选需导出的列，学号 默认勾选且不可取消。
选择保存路径后导出。
3. 列分类管理
点击 列分类。
可新增、重命名、删除列大类。
可将任意列归属到指定大类，用于筛选和详情页分组。
4. 学生详情编辑
在表格中双击学生行。
弹出详情窗口，按分类分标签显示字段，支持修改保存。
